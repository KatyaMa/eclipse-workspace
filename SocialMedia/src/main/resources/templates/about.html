ABOUT ME

Social Media for Kids

This is a quiz game built using HTML, CSS and JavaScript. The game is designed to be easy for kids to play. Features: - 10 questions are fetched from Open Trivia Database (opentdb.com) API with difficulty set to 'easy' and category set to 'animals' - A player can select only one answer for each question - The score is displayed at the end of the quiz and a play again button is provided
